# Key WOE Binning Plots

Contains selected WOE (Weight of Evidence) binning plots for the most important variables in the credit risk model.

The variables included are:

* status
* duration
* credit_history
* amount
* savings

These were chosen based on their high Information Value (IV), indicating strong predictive power for default risk.

## How to read the plots

* Bars show the distribution of good vs bad borrowers across bins
* The blue line shows the probability of default
* Each plot highlights how risk changes with different levels of the variable

## Notes

Only the most relevant variables are included to keep the analysis focused and avoid noise from low-impact features.
