# Model Performance Plots

This folder contains key plots used to evaluate the credit risk model.

## ROC Curve

The ROC curve shows how well the model distinguishes between default and non-default borrowers.
A curve closer to the top-left indicates better performance.

## Credit Score Distribution

This histogram shows the distribution of generated credit scores across borrowers.
It provides an overview of how applicants are spread across low and high risk segments.

## Notes

Together, these plots give a quick view of both model performance (ROC) and the final scoring output (score distribution).
